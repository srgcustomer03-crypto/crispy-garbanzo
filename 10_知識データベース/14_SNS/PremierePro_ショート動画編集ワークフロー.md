## 2. 概要（Executive Summary）
ショート動画市場に参入する**動画編集者およびクリエイター向けの「Premiere Pro 入門・効率化完全ガイド」**。
「バズるかどうかは量とスピードが命」という前提のもと、クオリティを維持しながら爆速で納品・投稿するための**具体的な環境設定とテクニック**を公開しています。

## 3. 実践的テクニック詳説
### Phase 1: 縦型シーケンスの最適化
*   **解像度:** 1080 x 1920 (9:16)
*   **フレームレート:** 基本的に30fps（または60fps）
*   **プリセット保存:** 毎回設定しなくて済むよう、この設定をプリセットとして保存することを推奨。

### Phase 2: ワークスペースの改造
縦型動画編集は、横長モニターの無駄なスペースが多くなりがち。
*   **パネル配置:** 「プロパティ」と「エフェクトコントロール」をアクセスしやすい位置（例：左上や右上）に固定し、タイムラインを長く確保するレイアウトに変更。
*   **ショートカット:** よく使う操作（カット、リップル削除）を左手側に集約する。

### Phase 3: セーフマージン（Safe Zone）の徹底
ショート動画は、画面上下や右側に「いいねボタン」「タイトル」「コメント欄」などのUIが被る。
*   **ガイド線:** UI被りゾーン（Dead Zone）を避けるためのガイドラインを引く。これがないと、重要なテロップがアイコンの下に隠れて読めなくなる。
*   **配置ルール:** テロップは中央〜やや上、重要な画は中央に配置。

### Phase 4: 時短編集術
1.  **アグレッシブなカット:** 「えー」「あー」などの間は全て削除（ジェットカット）。息継ぎすら消す勢いでテンポを上げる。
2.  **テロップの一括スタイル適用:**
    *   数百ある字幕レイヤーを一つ一つ弄るのではなく、リンクスタイルやラベル機能を使い、一括でフォント・色・サイズを変更する。
3.  **音量調整:** オーディオゲインで全体を底上げし、コンプレッサーで音圧を揃える工程をバッチ処理化。

## 4. 視覚的ポイント（Visual Analysis）
*   **ワークスペース画面:** 実際のPremiere Proの画面レイアウト（縦長のプレビュー画面とパネル配置）が詳細に紹介されている。
*   **セーフゾーン:** 実際にどこまでがUIで隠れるかを示すオーバーレイ画像が登場する。
