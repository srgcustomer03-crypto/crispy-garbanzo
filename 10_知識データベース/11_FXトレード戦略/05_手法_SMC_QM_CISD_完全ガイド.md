# SMC・QM・CISD 完全ガイド

**概要**
トレンド転換やトレンド継続の初動を1分足レベルで精密に捉える「YDHS手法」の核心的ロジック。
SMC (Smart Money Concept) の用語を用いているが、実践的なプライスアクション重視のスタイル。

## 構成要素

### 1. QM (Quasimodo Pattern)
*   **定義:** 高値（安値）を更新した後、直近の安値（高値）を割り込む動き。ダイバージェンスを伴う強力な反転パターン。
*   **活用:** エントリーの「環境」を認識するために使用。QMが発生＝トレンド転換の可能性大。

### 2. CISD (Change in State of Delivery)
*   **定義:** 価格配信の状態変化。
*   **判定:** 実体でのラインブレイクが必須。ヒゲだけのブレイクはダマシの可能性。
*   **役割:** トレンド転換の「確定」シグナル。これが決定打となる。

### 3. OC (Order Candle) / OB (Order Block)
*   **定義:** エントリートリガーとなる「注文が入っているローソク足」。
*   **活用:** QM/CISD発生後の **リテスト（戻り）** 目標。ここまで価格を引きつけてエントリーする。

### 4. Trendline (トレンドライン)
*   **活用:** 上昇/下降トレンドラインのブレイクが、CISDの予兆または確定として機能する。

### 5. The 1-2-3 Pattern (波動カウント)
*   **ロジック:**
    *   **1 (First Drive):** QM/CISDを伴う最初の強い動き。
    *   **2 (Correction/Retest):** OCやQMラインへの戻り。**ここがエントリーポイント。**
    *   **3 (Extension):** トレンド方向への伸び。
    *   師匠曰く「2はqm.cisdから」。第2波を狙うのが最もリスクリワードが良い。

## エントリー手順 (セットアップ)
1.  **待つ:** 自分の都合で入らない。「条件が揃う」のを待つ。
2.  **確認:** QM形成 → トレンドライン/CISDブレイクを確認。
3.  **引きつけ:** 価格が戻ってくるのを待つ (Retest)。
4.  **実行:** `QM` + `CISDライン` + `OC` (+ `14:00`などの時間優位性) が重なる **Confluence (合流点)** でエントリー。
